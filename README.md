# android-retrofit
Retrofit implementation for GET, POST, PUT and PATCH requests

REST API used 
https://jsonplaceholder.typicode.com/

Routes<br>
GET	/posts <br>
GET	/posts/1 <br>
GET	/posts/1/comments<br>
GET	/comments?postId=1 <br>
GET	/posts?userId=1 <br>
POST	/posts<br>
PUT	/posts/1 <br>
PATCH	/posts/1<br>
DELETE	/posts/1<br>
