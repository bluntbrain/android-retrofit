# android-retrofit
Retrofit implementation for GET, POST, PUT and PATCH requests

REST API used 
https://jsonplaceholder.typicode.com/

All HTTP methods are supported.

GET	/posts
GET	/posts/1
GET	/posts/1/comments
GET	/comments?postId=1
GET	/posts?userId=1
POST	/posts
PUT	/posts/1
PATCH	/posts/1
DELETE	/posts/1
